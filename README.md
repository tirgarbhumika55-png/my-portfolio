## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/tirgarbhumika55-png/my-portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
